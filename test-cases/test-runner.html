Activity Viewer Test Suite

Manual test cases for the Plain Text Fitness viewer. Click each link to test the functionality.

Test 1: Full Activity MANUAL

Scenario: Complete activity with GPS data (.fit + .gpx) and metadata

Open Test →

Test Checklist:

  • ✓ Page loads without errors
  • ✓ Activity title shows "Morning Run"
  • ✓ Date and activity type displayed
  • ✓ Map shows route with green start marker and red finish marker
  • ✓ Three charts displayed: Elevation, Heart Rate, Pace
  • ✓ Stats cards show: Distance, Duration, Avg Pace, Elevation Gain, Heart Rate
  • ✓ Hovering over charts shows vertical crosshair on ALL charts
  • ✓ Hovering over charts shows blue marker on map
  • ✓ km/mi toggle works and updates all values
  • ✓ Chart tooltips show distance with 2 decimal places
  • ✓ Files detected shown (activity.fit, activity.gpx, metadata.yaml)

Test 2: Metadata Only MANUAL

Scenario: Non-GPS activity (gym workout) with only metadata

Open Test →

Test Checklist:

  • ✓ Page loads without errors
  • ✓ Activity title shows "Gym Session - Upper Body"
  • ✓ Date shows "Sunday, November 3, 2025"
  • ✓ Activity type shows "strength-training"
  • ✓ Description displays workout details (bench press, pull-ups, etc.)
  • ✓ NO map displayed
  • ✓ NO charts displayed
  • ✓ NO stats cards displayed
  • ✓ NO km/mi toggle displayed
  • ✓ Files detected shows only metadata.yaml

Test 3: Activity with Media MANUAL

Scenario: Activity with GPS data and photos in media/ folder

Open Test →

Test Checklist:

  • ✓ All features from Test 1 work
  • ✓ Activity title shows "Trail Run with Photos"
  • ✓ Description card is visible
  • ✓ Three 64x64 thumbnails displayed below description
  • ✓ Thumbnails show photo1.svg, photo2.svg, photo3.svg
  • ✓ Clicking thumbnail opens full-screen modal gallery
  • ✓ Modal shows full-size image
  • ✓ ESC key closes modal
  • ✓ Left arrow (←) navigates to previous image
  • ✓ Right arrow (→) navigates to next image
  • ✓ Previous/Next buttons work
  • ✓ Close button (×) works
  • ✓ Gallery wraps around (last → first, first → last)

Instructions

To run these tests:

  1. Start a local web server from the project root: python3 -m http.server 8000
  2. Open this page: http://localhost:8000/test-cases/test-runner.html
  3. Click each "Open Test →" link
  4. Go through the checklist for each test
  5. Verify all items pass

Note: Directory listing must be enabled on your web server for media detection to work in Test 3.
